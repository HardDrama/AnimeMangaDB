## Fixture Naming

Episode fixtures must follow:

episode_XXX.html

Examples:

episode_001.html
episode_148.html
episode_293.html

Three-digit numbering is required.

## Capture Requirements

Fixtures must be saved as the complete HTML document.

Do not trim HTML.

Do not preprocess HTML.

Do not modify whitespace.

Characterization tests define the observed structure from the captured source.
