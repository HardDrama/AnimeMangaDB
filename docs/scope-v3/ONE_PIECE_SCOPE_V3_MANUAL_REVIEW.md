## Review Instructions

For every selected record:

1. Open the stored `source_url`.
2. Confirm the page corresponds to the expected chapter number.
3. Compare the stored chapter title with the page’s `Viz Title`.
4. Compare the stored manga arc with the verified arc category.
5. Mark each field as:
   - PASS
   - FAIL
   - NOT APPLICABLE
6. Record any source anomaly or ambiguity in Notes.

Do not modify database records during this review.
