# AnimeMangaDB

AnimeMangaDB (working title) is an open-source project that maps anime episodes to their corresponding manga chapters.

## Goals

- Scrape supported anime sources
- Store episode/chapter relationships
- Provide a REST API
- Build a searchable website

## Planned Tech Stack

- Python
- FastAPI
- SQLAlchemy
- PostgreSQL
- React
- TypeScript

## Project Status

- In Development
